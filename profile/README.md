# DeloneCommons

Scientific Python tools for computational geometry of periodic structures.

DeloneCommons develops reusable packages for crystallography, molecular crystals, periodic graphs, and Voronoi/Laguerre methods. The focus is on small, composable tools that can support larger workflows in computational chemistry, structural analysis, and materials modeling.

## Repositories

| Repository | Status | Purpose |
|---|---:|---|
| [`pyvoro2`](https://github.com/DeloneCommons/pyvoro2) | beta | Python interface for 2D/3D Voronoi and power/Laguerre tessellations, including periodic-boundary support and topology utilities |
| [`pbcgraph`](https://github.com/DeloneCommons/pbcgraph) | alpha | Translation-labeled periodic graphs on integer lattices, with exact infinite-lift connectivity semantics |
| [`atomref`](https://github.com/DeloneCommons/atomref) | usable | Curated atomic reference data for geometry and structure-analysis workflows |
| [`atomref-proatoms`](https://github.com/DeloneCommons/atomref-proatoms) | usable | Reproducible spherical proatomic radial electron-density datasets for atom-centered theoretical-chemistry and crystallographic models |
| [`powerfit`](https://github.com/DeloneCommons/powerfit) | research | Reproducible materials for inverse fitting of power-diagram weights from separator constraints |

## Scope

The organization is centered on computational geometry and periodic-structure infrastructure, with crystallographic applications as the main scientific driver.
